# kore-fingerprint v0.1.0 🎉 首发版本

> MCP-controllable anti-detect browser for AI multi-account automation.
> AI 多账号自动化的 MCP 反检测浏览器。

## ✨ 核心特性

- **MCP 协议原生支持** - stdio + SSE 双传输
- **7 个开箱即用工具** - navigate / screenshot / click / type / extract / get_fingerprint / solve_captcha
- **真实 Puppeteer 集成** - puppeteer-extra + stealth plugin
- **OCR + Vision 双引擎** - 本地 Tesseract + 远端 Hermes Vision 兜底
- **轻量资源占用** - 3.6G 内存服务器，Chromium 仅 85MB RSS
- **跨平台打包** - Linux AppImage / Windows exe / macOS dmg

## 📦 安装

### 方式 1: 下载安装包

从 Releases 页面下载对应平台安装包:
- KoreFingerprint-0.1.0-linux-x86_64.AppImage (304 MB)
- KoreFingerprint-0.1.0-windows-x64.exe
- KoreFingerprint-0.1.0-mac.dmg

### 方式 2: 源码编译

```bash
git clone https://github.com/kore-01/kore-fingerprint.git
cd kore-fingerprint
npm install
npm run build:linux  # 或 build:win / build:mac
```

## 🚀 快速开始

### 1. 启动 MCP Server

```bash
# stdio 模式（给 Claude Desktop 用）
node src/main/mcp/server.mjs

# SSE 模式（远程服务器）
KORE_MCP_MODE=sse KORE_MCP_PORT=39391 node src/main/mcp/server.mjs
```

### 2. 配置 Claude Desktop

参见 docs/claude-config/README.md

### 3. 在 Claude 中使用

> 帮我用 kore-fingerprint 打开 https://example.com，截屏给我看

## 📋 系统要求

- Linux: Ubuntu 22.04+, 3.6G RAM, libnss3 等 16 个系统库
- macOS: macOS 12+, 4G RAM
- Windows: Windows 10+, 4G RAM

Node.js: **>= 22.12.0**

## 🐛 已知问题

- 远程 SSE 端点的 /messages 需要先建立 SSE session 才能 POST（MCP SSE 协议规范）
- WebGL fingerprint 在 headless 模式下返回 undefined（Chromium 限制）
- 多 profile 并发启动限制：同一时间只启动一个 Chromium 实例
